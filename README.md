# AI Interview Practice & Feedback System

An AI-powered mock interview platform built using **Python, Streamlit, and Groq LLM APIs**.

## Features
- HR, Technical, and Managerial interview modes
- AI-generated follow-up questions
- Structured feedback (strengths, improvements, score)
- Realistic mock interview flow

## Tech Stack
- Python
- Streamlit
- Groq LLM API

## How to Run
```bash
pip install streamlit groq
streamlit run app.py
